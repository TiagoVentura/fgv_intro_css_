Divirtam-se!

Neste exercício, vocês irão criar um código completo de raspagem de dados. O nosso objetivo é coletar todas as informações da agenda de trabalho do Presidente Bolsonaro no ano de 2020 e 2021.

Vocês precisam:

  1. Escreve o código para coletar informações.

  2. Capturar a informação e armazenar em um data frame.

  3. Este data frame precisa ter as seguintes colunas: dia, hora, descrição da reunião.

  4. Printar no final do código um sumário do data frame, usando o glimpse.

Este é um banco de dados que vocês podem usar no trabalho final do curso!
