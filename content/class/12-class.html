Objetivos Gerais

Para a aula de hoje, eu solicitei a vocês que escolhessem alguns gráficos retirados de alguns dos meus artigos acadêmicos para que possamos reproduzir estas visualizações. Os objetivos desta aula são três:

  • Mostrar de forma integrada o processo de manipulação de dados + visualização.

  • Fazer uma breve revisão das aulas de manipulação de dados.

  • Treinar o que aprendemos de ggplot2.

Materiais.

Todo o material da aula de hoje está disponível nesta pasta.

A pasta está dividida em tres sub-pastas: código, dados e outputs. Como aprendemos no tutorial de fluxo de trabalho, a pasta possui um arquivo de Projeto de R (.Rproj). Abram este projeto, e o código para cada gráfico deve rodar sem problemas maiores em seus computadores.
